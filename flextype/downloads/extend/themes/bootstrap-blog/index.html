Bootstrap Blog

Twitter Bootstrap Blog theme for Flextype.

Sergey Romanenko

Features

  • Fully Responsive
  • Cross browser compatible
  • Powered by Twitter Bootstrap Framework

Installation

  1. Download & Install all required dependencies.
  2. Create new folder /project/themes/bootstrap-blog
  3. Download Bootstrap Blog Theme and unzip theme to the folder /project/themes/bootstrap-blog
  4. Update setting theme with new value bootstrap-blog in /project/config/plugins/site/settings.yaml

The theme by itself is useful, but you may have an easier time getting up and running by installing a project demo content for this theme, it is placed in the _project folder. The Bootstrap Blog Project demo content contains: sample content, fieldsets for Admin Panel Plugin. Just copy _project folder content into your project folder.
